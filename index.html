Absolute positioning is positioned relative to closest positioned ancestor, if any. Otherwise it is placed relative to the initial containing block.

Relative is positioned according to normal flow of the document, and then offset relative to itself based on the values of top, right, bottom and left.

Eg:
Absolute
Relative

An element with position:fixed; is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled

Static positioning is the default. It defines the position of a given box essentially as an unpositioned element- it flows in the normal rendering sequence of the web page.
